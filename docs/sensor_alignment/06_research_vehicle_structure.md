# 6. 연구용 차량 권장 구조

## 6.1 연구용 차량의 특성

연구용 자율주행 차량은 양산차와 달리 다음 특성을 가진다.

| 항목 | 연구용 차량 | 양산차 |
|------|-----------|------|
| 센서 구성 변경 | 빈번 (실험마다 변경 가능) | 고정 |
| Calibration 빈도 | 높음 | 공장 1회 + 딜러 점검 |
| 제작 수량 | 1~수대 | 수만~수백만 대 |
| 예산 | 제한적 | 대규모 |
| 요구 내구성 | 연구 기간 (수년) | 차량 수명 (10년+) |
| 정비 인력 | 연구자 직접 | 전문 서비스센터 |

## 6.2 권장 구조 설계 원칙

### 6.2.1 모듈형 레일 시스템

하나의 공통 레일(Rail)을 차량 루프·전후방에 설치하고,
브라켓을 레일에 탈착 가능하게 하는 구조를 권장한다.

```
[루프 레일 시스템 — 탑뷰]

     전방
  ┌─────────────────────────────┐
  │ ├─────────────────────────┤ │  ← 루프 레일 (좌우)
  │ │   [LiDAR]  [GNSS/IMU]  │ │
  │ ├─────────────────────────┤ │
  │                             │
  └─────────────────────────────┘
     후방

[전방 바 시스템 — 정면뷰]

  ├─────────────────────────────┤  ← 전방 크로스바
  │    [CAM-02]   [CAM-01]     │
  │                             │
  │        차량 전방             │

레일/바 재질: 알루미늄 프로파일 (40×40 mm 또는 30×30 mm)
마운트: T-너트 + 볼트 슬라이딩
```

### 6.2.2 빠른 탈착 구조 (Quick-Release)

- 일상 운용: 볼트 체결 + Dowel Pin으로 완전 고정
- 센서 교체: 볼트 2~4개 해제만으로 센서 분리 가능
- 레일 슬라이딩: 공구 없이 위치 조정 가능 (개략 위치)
- 정밀 위치: Dowel Pin으로 최종 확정

### 6.2.3 인터체인지블 마운트 (Interchangeable Mount)

동일 OAK-D Pro PoE 카메라를 다른 위치에 이동할 경우를 대비한다.

```
카메라 본체 ← M4 × 4개 → 카메라 어댑터 플레이트
                                    ↕
                          위치별 마운트 브라켓
                          (Type-A / Type-B)
                                    ↕
                              레일 / 차량 구조물
```

어댑터 플레이트를 공통화하면 카메라 본체를 분리 없이 브라켓 통째로 이동 가능.

## 6.3 권장 장착 위치

### 6.3.1 전방 시스템

```
[전방 — 측면뷰]

        ──────────────────────  루프
         │
         │ 전방 지주 (A-Pillar 또는 전방 프레임)
         │
  ──────┤├────────────────────  전방 크로스바 (H = 1.4 ~ 1.8 m)
        │CAM-02 (원거리)│
        │CAM-01 (근거리)│
  ─────────────────────────────  범퍼 상단 (H = 0.6 ~ 0.9 m)
```

크로스바 위치: 지면에서 1.4 ~ 1.8 m 사이 (시야·공기저항 균형)

### 6.3.2 후방 시스템

전방 시스템과 동일한 구조를 후방에 미러로 배치한다.

### 6.3.3 측방 카메라

```
[우측 — 측면뷰]

  ──────────────────────────  루프
  │
  │  ← 측방 마운트 암 (100~150 mm 돌출)
  │         [CAM-04]
  │
  ──────────────────────────  도어 상단 (H ≈ 1.3 m)
```

측방 카메라는 도어 상단 또는 사이드미러 하단에 마운트.
돌출 길이는 최소화하고 리브로 강성 확보.

### 6.3.4 LiDAR (Livox Mid-360)

```
[루프 — 탑뷰]

  ┌──────────────────────────────┐
  │                              │
  │         [Mid-360]            │  ← 루프 중앙
  │                              │
  └──────────────────────────────┘

장착 높이: 루프 최상단 (지면 약 1.7 ~ 2.0 m)
이유: 360° 시야 확보, 차량 본체에 의한 음영 최소화
```

### 6.3.5 GNSS 안테나

```
루프 후단 중앙 (차량 중심선 상)
- 가능한 하늘이 트인 위치
- LiDAR와 최소 300 mm 이격 (전파 간섭 방지)
- GNSS 안테나 케이블 최대 굽힘 반경: 50 mm 이상
```

### 6.3.6 IMU

```
차량 중앙 하부 또는 차체 플로어
- 차량 무게중심 근처 권장
- 진동 경로와 직접 접하지 않도록 고무 댐퍼 삽입
- 전자기기(인버터, 모터)와 최소 300 mm 이격
```

## 6.4 전기/배선 관리

### 6.4.1 PoE 카메라 배선

```
[카메라] ─ Cat6e ─ [PoE Switch]
                       │
                   [연산 서버]

PoE Switch 위치: 루프 또는 트렁크 (카메라 케이블 길이 최소화)
각 카메라별 케이블 길이: 3 m 이내 권장 (PoE+ 전압 강하 고려)
케이블 루팅: 차량 내부 패널 경유, 방수 그로밋 처리
```

### 6.4.2 LiDAR 배선

```
[Livox Mid-360] ─ M12 커넥터 ─ [전원/이더넷 분배기] ─ [연산 서버]
전원: 12 V DC (별도 차량 배선 필요)
이더넷: Cat6e, 10 m 이내
```

## 6.5 Calibration 작업 공간 확보

차량 전방/후방에 Calibration 보드를 세울 공간을 고려한 장착 위치를 선택한다.

```
[전방 Calibration 셋업]

[Checkerboard / AprilTag 보드]
         ↑
         │ 2 ~ 5 m
         ↓
    [전방 카메라]   [LiDAR]
       [차량 전방]
```

- 보드 최소 거리: 0.5 m (근거리 카메라)
- 보드 최적 거리: 1.5 ~ 3.0 m (정밀도 최적)
- 보드 크기: 카메라 시야의 30~60% 점유 권장
