# 3. Camera Bracket 설계안

## 3.1 카메라 목록 및 장착 위치

| ID | 명칭 | TF 링크 | 위치 | 주요 역할 |
|----|------|---------|------|---------|
| CAM-01 | 전방 근거리 | `cam_front_center_near` | 전방 중앙 하단 | 교차로 인식, 보행자 근거리 |
| CAM-02 | 전방 원거리 | `cam_front_center_far` | 전방 중앙 상단 | 고속 전방 물체, 차선 원거리 |
| CAM-03 | 좌후방 | `cam_rear_left` | 좌측 후방 | 차선 변경, 후방 좌측 사각지대 |
| CAM-04 | 우후방 | `cam_rear_right` | 우측 후방 | 차선 변경, 후방 우측 사각지대 |
| CAM-05 | 후방 근거리 | `cam_rear_center_near` | 후방 중앙 하단 | 주차, 후방 근접 물체 |
| CAM-06 | 후방 원거리 | `cam_rear_center_far` | 후방 중앙 상단 | 후방 차량 원거리 추적 |

## 3.3 전방 카메라 브라켓 (CAM-01 / CAM-02)

전방에는 근거리·원거리 2개의 카메라를 수직으로 배치한다.

```
[전방 브라켓 — 전면 뷰]

        ┌──────────────────────────┐
        │     [CAM-02 원거리]       │  ← 상단 마운트 (High)
        │                          │
        ├──────────────────────────┤  ← 높이 슬롯 (±20 mm)
        │                          │
        │     [CAM-01 근거리]       │  ← 하단 마운트 (Low)
        └──────────────────────────┘
                 차량 전방 방향 →

[전방 브라켓 — 측면 뷰]

      CAM-02 (원거리)
        ↑ Pitch ─5° (원거리 — 지평선 기준 약간 하향)
        │
   ─────●─────  힌지축
        │
   [베이스 플레이트]
        │
   ─────●─────  차량 구조물 (Datum면)
        ↓
      CAM-01 (근거리)
        ↑ Pitch +0° (수평)
```

### 3.3.1 CAM-01 (전방 근거리) 초기 정렬값

| 항목 | 초기값 | 비고 |
|------|-------|------|
| Pitch | 0° | 지면 수평 |
| Yaw | 0° | 차량 전방 정렬 |
| Roll | 0° | |
| Height | 차량별 측정 필요 | 범퍼 상단 기준 |

### 3.3.2 CAM-02 (전방 원거리) 초기 정렬값

| 항목 | 초기값 | 비고 |
|------|-------|------|
| Pitch | -3° ~ -5° | 원거리 주시 (하향) |
| Yaw | 0° | 차량 전방 정렬 |
| Roll | 0° | |
| Height | CAM-01 상단 + 60 mm 이상 | 겹침 방지 |

## 3.4 후방 카메라 브라켓 (CAM-05 / CAM-06)

전방 브라켓과 동일한 구조를 미러(좌우 반전)로 사용한다.

```
[후방 브라켓 — 후면 뷰]

        ┌──────────────────────────┐
        │     [CAM-06 원거리]       │  ← 상단
        │                          │
        ├──────────────────────────┤
        │     [CAM-05 근거리]       │  ← 하단
        └──────────────────────────┘
                 차량 후방 방향 ←
```

- CAM-06 Pitch: -3° ~ -5° (후방 원거리 하향)
- CAM-05 Pitch: 0° (수평)
- 전방 브라켓과 동일 부품 공용 가능 (설계 단순화)

## 3.5 측방 카메라 브라켓 (CAM-03 / CAM-04)

```
[측방 브라켓 — 탑뷰]

차량 전방 →

   ┌──────────────────┐
   │ 차량 본체         │
   │                  │
   │        [CAM-03]──●──→ 후방 좌측 (Yaw −120° ~ −150°)
   │        [CAM-04]──●──→ 후방 우측 (Yaw +120° ~ +150°)
   │                  │
   └──────────────────┘

[측방 브라켓 — 측면뷰]

         [CAM-03 또는 CAM-04]
              ↑ Pitch -10° ~ -20° (후방 노면 시야 확보)
         ─────●─────  힌지축
         [차량 측면 구조물]
```

### 3.5.1 측방 카메라 초기 정렬값

| 항목 | 좌후방 (CAM-03) | 우후방 (CAM-04) |
|------|---------------|---------------|
| Pitch | -15° | -15° |
| Yaw | -135° (차량 좌후방) | +135° (차량 우후방) |
| Roll | 0° | 0° |

## 3.6 브라켓 CAD 설계 고려사항

### 3.6.1 강성

- 브라켓 주재질: 알루미늄 6061-T6
  - 경량(2.7 g/cm³), 가공 용이, 부식 저항
- 두께: 베이스 플레이트 4 mm 이상, 마운트 암 3 mm 이상
- 보강 리브(Rib): 마운트 암 길이 > 50 mm 이면 삼각 리브 추가

### 3.6.2 진동

- 고유진동수(Natural Frequency) 목표: 50 Hz 이상
  (차량 엔진/도로 진동: 주로 5~30 Hz)
- 브라켓 설계 후 FEA 또는 간이 타격 검사(Tap Test) 실시 권장
- 진동 전달 억제 필요 시: 고무 댐퍼(Rubber Damper) 삽입

### 3.6.3 소재 및 제작

| 항목 | 권장 | 비고 |
|------|------|------|
| 본체 | 알루미늄 6061-T6 | 강도/무게 균형 |
| 표면처리 | 아노다이징(Anodizing) | 내식성, 표면 경도 |
| 체결 볼트 | SUS304 | 이종금속 부식 방지 |
| 제작 방법 | CNC 밀링 | 연구용 소량 제작 |
| 공차 | ±0.1 mm | Dowel Pin 구멍 H7 공차 |

### 3.6.4 케이블 관리

- 카메라 PoE 케이블 루팅 경로를 브라켓 설계 시 함께 고려한다.
- 케이블 클램프(Cable Clamp) 장착 홀을 브라켓에 포함한다.
- 케이블 최소 굽힘 반경: 37.5 mm 이상 (Cat6 기준)
- 진동에 의한 케이블 피로 방지를 위해 여유 길이 확보 (100 mm 이상)

## 3.7 브라켓 공용화 전략

연구 비용 절감과 유지보수 편의를 위해 부품 공용화를 추구한다.

| 브라켓 타입 | 사용 카메라 | 비고 |
|-----------|-----------|------|
| Type-A (전후방 공용) | CAM-01, CAM-02, CAM-05, CAM-06 | 좌우 대칭 설계 |
| Type-B (측방) | CAM-03, CAM-04 | 좌우 미러 설계 |

Type-A 1종, Type-B 1종 설계로 총 2종 브라켓으로 모든 카메라 커버 가능.
