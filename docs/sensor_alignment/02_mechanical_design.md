# 2. Mechanical Alignment 설계

## 2.1 설계 요구사항

| 항목 | 요구사항 |
|------|---------|
| Pitch 조절 범위 | ±15° |
| Yaw 조절 범위 | ±15° |
| Roll 미세조정 | ±5° |
| Height 조정 | 슬롯 방식 ±20 mm |
| 반복 조립 정밀도 | Dowel Pin 기준 ±0.1 mm 이내 |
| 진동 내성 | 비포장 저속 주행(≤30 km/h) 조건에서 각도 변화 ±0.05° 이내 |

## 2.2 좌표계 정의

ROS2 REP-105 기준을 따른다.

```
X : 전방 (Forward)
Y : 좌방 (Left)
Z : 상방 (Up)

Pitch : Y축 회전 (카메라 상하각)
Yaw   : Z축 회전 (카메라 좌우각)
Roll  : X축 회전 (카메라 기울기)
```

## 2.3 기준면(Datum) 설계

Datum은 브라켓이 차량 구조물에 장착될 때 위치를 결정하는 기준이 되는 면이다.

### 2.3.1 1차 Datum (Primary Datum) — Z 방향 구속

- 차량 루프 또는 전후방 구조물의 평탄면을 1차 Datum으로 사용한다.
- 브라켓 베이스 플레이트 하면이 이 면에 완전히 밀착되어야 한다.
- 면 평탄도 요구사항: 0.3 mm 이내 (현장 측정 기준)

### 2.3.2 2차 Datum (Secondary Datum) — X 또는 Y 방향 구속

- 차량 센터라인 또는 기준 프레임 측면을 2차 Datum으로 사용한다.
- 브라켓에 기준 엣지(Reference Edge)를 두어 정렬한다.

### 2.3.3 위치결정핀 (Dowel Pin)

```
             [브라켓 베이스 플레이트]
             ┌──────────────────────┐
             │  ○  ←── Dowel Pin   │
             │                      │
             │         ●            │  ← 카메라 중심
             │                      │
             │  ○  ←── Dowel Pin   │
             └──────────────────────┘
                    [차량 구조물]
```

- 핀 직경: M6 또는 Ø6h7 (공차 h7 = 0 / -0.012 mm)
- 핀 구멍 공차: H7 (0 / +0.012 mm)
- 핀 재질: SUS304 (녹 방지)
- 배치: 브라켓 대각선 방향 2개소, 가능한 멀리 배치하여 각도 오차 최소화

### 2.3.4 슬롯 홀 (Slot Hole)

- 고정 볼트 체결 전 미세 조정을 위한 슬롯을 둔다.
- Yaw 조정용: 수직 방향 원호 슬롯 (R = 브라켓 회전 반경)
- Pitch 조정용: 수평 방향 원호 슬롯
- 슬롯 폭: 체결 볼트 직경 + 1 mm 여유

## 2.4 자유도(DOF) 조정 구조

### 2.4.1 Pitch 조절

카메라 마운트 플레이트와 베이스 플레이트 사이에 힌지 축을 두어 Pitch를 조절한다.

```
[사이드뷰]

      ┌────┐  ← 카메라 마운트
      │ CAM│
      └──┬─┘
         │ ← 힌지축 (Y축)
    ─────●─────  ← 베이스 플레이트
         │
    [차량 구조물]
```

- 조절 방식: 캡스크류(Cap Screw) 풀림 → 각도 조정 → 체결
- 각도 표시: 베이스에 각도 눈금 마킹 (1° 단위)
- 잠금: M5 × 2개 + 스프링 와셔

### 2.4.2 Yaw 조절

베이스 플레이트와 차량 장착부 사이에 회전 슬롯 구조를 둔다.

```
[탑뷰]

  ← Yaw 슬롯 ─→
  ┌─────────────┐
  │  ○──────○  │  ← 원호 슬롯
  │      ▲      │
  │    [CAM]    │
  └─────────────┘
```

- 조절 방식: 슬롯 내 볼트 슬라이딩
- 범위: ±15°
- 각도 표시: 슬롯 외측에 각도 눈금

### 2.4.3 Roll 미세조정

카메라 본체와 마운트 사이에 O링 또는 마찰면을 이용한 회전 조정을 둔다.

- 조절 방식: 카메라 체결 볼트 미세 해제 후 회전
- 범위: ±5°
- 잠금: M4 × 4개 균등 배치

### 2.4.4 Height 조정

베이스 플레이트에 수직 슬롯을 두어 높이 조정을 허용한다.

- 범위: ±20 mm
- 고정: M6 볼트 × 2개 + 평와셔 + 스프링 와셔

## 2.5 고정 방식 (Locking)

| 단계 | 방식 | 목적 |
|------|------|------|
| 조정 중 | 볼트 반체결 | 슬라이딩 허용 |
| 캘리브레이션 전 | 손으로 잠금 | 위치 임시 고정 |
| 최종 고정 | 규정 토크 체결 | 진동 중 풀림 방지 |
| 장기 운용 | 나사잠금제 (Loctite 243) | 볼트 자연 풀림 방지 |

### 체결 토크 기준

| 볼트 규격 | 재질 | 권장 토크 |
|----------|------|---------|
| M4 | SUS304 | 2.0 N·m |
| M5 | SUS304 | 4.0 N·m |
| M6 | SUS304 | 7.0 N·m |

## 2.6 열팽창 고려

알루미늄(브라켓)과 철(차량 구조물)의 열팽창 계수 차이로 인해
온도 변화 시 미세한 위치 변화가 발생할 수 있다.

| 재질 | 열팽창 계수 |
|------|-----------|
| 알루미늄 (6061-T6) | 23.6 μm/m·°C |
| 철 / 강 | 11.7 μm/m·°C |

100 mm 길이 브라켓에서 온도 20°C 변화 시 약 0.12 mm 차이 발생.
이는 카메라 위치 기준으로 Yaw 약 0.07° 에 해당하며,
Software Calibration 허용 범위 내에 포함되므로 실용상 무시 가능하다.

단, 장기간 직사광선 노출 환경에서는 주기적 재검증을 권장한다.
