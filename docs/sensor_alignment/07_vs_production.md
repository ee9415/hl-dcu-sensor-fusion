# 7. 양산차 대비 연구용 차량의 설계 차이점

## 7.1 개요 비교

| 항목 | 양산차 | 연구용 차량 |
|------|-------|-----------|
| 센서 종류 | 표준화 (ADAS 카메라, 레이더) | 다종 센서 (LiDAR, PoE 카메라, IMU 등) |
| 센서 위치 | 설계 고정 | 실험마다 변경 가능 |
| 브라켓 | 사출성형/다이캐스팅 전용 부품 | CNC 가공 범용 브라켓 |
| Calibration | 공장 자동 Calibration 라인 | 수작업 + 소프트웨어 |
| 정렬 재현성 | 조립 지그(Jig) 기반 0.01 mm 정밀 | Dowel Pin 기반 ±0.1 mm |
| 내구성 | 차량 수명 (10년, 20만 km) | 연구 기간 (2~5년) |
| 생산 수량 | 수만~수백만 대 | 1~수대 |
| 비용 | 부품당 단가 낮음 (대량 생산) | 부품당 단가 높음 (소량 제작) |

## 7.2 기계 구조

### 양산차 브라켓

```
[양산차 카메라 마운트 (예: 전방 카메라)]

  ┌─────────────────────┐
  │     루프라인 글라스   │
  │  ┌─────────────────┐│
  │  │  전용 사출 브라켓 ││  ← 차량 구조에 맞춤 설계
  │  │     [카메라]     ││      Calibration 없이 장착 즉시 사용
  │  └─────────────────┘│      교체 시 딜러에서만 가능
  └─────────────────────┘
```

특징:
- 조립 지그로 0.01 mm 이내 정밀 재현
- 각도 조정 불가 (고정 설계)
- 센서 종류 변경 불가
- 공장 Calibration 데이터 ECU 내장

### 연구용 차량 브라켓

```
[연구용 조정형 브라켓]

  ─────────────────────────  레일
       │
  ─────●─────  Dowel Pin (위치 재현)
  [베이스 플레이트]
       │
  ─────●─────  힌지 (Pitch 조정)
  [마운트 플레이트]
       │
  ─────●─────  슬롯 (Yaw 조정)
    [카메라]
```

특징:
- 조정 자유도 3축 (Pitch / Yaw / Roll)
- Dowel Pin으로 반복 재장착 위치 재현
- 센서 종류 교체 가능 (어댑터 플레이트)
- 현장 Calibration 필수

## 7.3 Calibration 방식

### 양산차 Calibration

```
공장 Calibration 라인
        │
   ┌────┴────┐
   │ 자동화  │
   │ 지그    │  ← 전 차종 통일 지그
   └────┬────┘
        │
   ECU 자동 저장  ← 작업자 개입 최소
        │
   소비자 납차
        │
   딜러 서비스:
   - 전면유리 교체 후 재Calibration (ADAS 카메라)
   - 전용 보정 타겟 + OBD 연결 자동화 절차
```

### 연구용 차량 Calibration

```
연구자 수작업 정렬
        │
   Checkerboard / AprilTag 촬영
        │
   ROS2 패키지 실행 (kalibr, camera_calibration)
        │
   YAML 파라미터 수동 확인
        │
   Launch 파일 반영
        │
   RViz 시각적 검증
```

**핵심 차이**: 양산차는 자동화·표준화, 연구용 차량은 유연성·반복가능성 우선.

## 7.4 진동·내구 설계 기준

| 항목 | 양산차 | 연구용 |
|------|-------|-------|
| 진동 시험 기준 | ISO 16750-3, AEC-Q100 | 자체 기준 (논문/학회 수준) |
| 충격 내성 | 50g 이상 | 10~20g (연구 환경) |
| 온도 범위 | -40°C ~ +85°C | -10°C ~ +60°C (국내 환경) |
| IP 등급 | IP67 이상 | IP54 이상 (야외 운용 최소) |
| 수명 사이클 | 10년 / 20만 km | 2~5년 / 수만 km |

## 7.5 비용 구조

| 항목 | 양산차 | 연구용 |
|------|-------|-------|
| 브라켓 단가 | 수백 원 ~ 수천 원 (대량) | 수만 원 ~ 수십만 원 (소량 CNC) |
| Calibration 비용 | 라인 투자 후 대당 낮음 | 연구자 시간 + 장비 비용 |
| 센서 가격 | 대량구매 할인 | 소량 정가 구매 |
| 유지보수 | 딜러 네트워크 | 연구자 직접 |

## 7.6 연구용 차량이 양산차에서 참고할 수 있는 개념

| 양산차 개념 | 연구용 적용 방안 |
|-----------|--------------|
| 조립 지그 | Dowel Pin + 기준면 → 재현성 확보 |
| 전용 사출 브라켓 | 반복 사용 가능한 CNC 알루미늄 브라켓 |
| 공장 Calibration 자동화 | 반자동 스크립트 (단일 명령어 실행) |
| Calibration 데이터 ECU 저장 | YAML 파일 버전 관리 (Git) |
| 전담 점검 주기 | 주행 거리·시간 기반 검증 주기 수립 |

## 7.7 연구 성과의 양산 이전 가능성

연구용 플랫폼에서 검증된 센서 배치 및 Calibration 방법은 다음 경로로 양산에 이전 가능하다.

```
연구용 플랫폼 (유연·조정식)
        │
  최적 위치 실험적 확정
        │
  양산 전용 고정 브라켓 설계
        │
  공장 자동 Calibration 라인 구축
        │
  양산 차량 탑재
```

이 프로젝트(hl-dcu-sensor-fusion)는 이 경로의 첫 번째 단계에 해당한다.
